# Last
其实这个是优酷以前的那个环形菜单，那个时候自己剖析完之后决定重新做个另一种版本的玩一下，也就是现在这样。这也算是搞Android以来的第一个控件，虽然没什么太大用处，但很有纪念意义，让我想起当时2013年的自己，那会儿熬夜去弄这个东西的冲劲儿。最近翻以前的资料看到它，索性也上传上来吧。
